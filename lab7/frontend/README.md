# Sport Platform Frontend

Next.js интерфейс для платформы спортивных секций.

## Запуск

```bash
cp .env.example .env
npm install
npm run dev
```

Приложение ожидает API по адресу `NEXT_PUBLIC_API_URL`, по умолчанию `http://localhost:8080`.

## Команды

- `npm run dev` - режим разработки
- `npm run build` - production-сборка
- `npm run typecheck` - проверка TypeScript
- `npm run check` - Biome-проверка
